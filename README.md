# משחק למידה אינטראקטיבי במתמטיקה לכיתה י' 🎓📐

משחק למידה אינטראקטיבי, עשיר ויזואלית ומתקדם טכנולוגית, המותאם לתוכנית הלימודים במתמטיקה של כיתה י' (3, 4 ו-5 יחידות לימוד). המשחק משלב אלמנטים של משחוק (Gamification) כגון נקודות ניסיון (XP), כוכבים, רצף למידה יומי (Streak) ותגים מיוחדים כדי לעודד תלמידים לתרגל ולשלוט בחומר הלימודי בצורה מהנה וחווייתית.

---

## ✨ תכונות עיקריות

*   **עולמות תוכן מגוונים (Worlds):**
    1.  **אלגברה ופונקציות:** משוואות ריבועיות, חקירת פרבולות ומערכות משוואות.
    2.  **גיאומטריה וטריגונומטריה:** משפט פיתגורס, יחסים טריגונומטריים במשולש ישר זווית ודמיון.
    3.  **אנליזה וחשבון דיפרנציאלי:** מבוא לנגזרת, חוקי גזירה בסיסיים וחישוב שיפוע משיק.
*   **מגוון סוגי שאלות אינטראקטיביות:**
    *   **שאלה מספרית (Numerical):** הזנת תשובה מספרית מדויקת (תומך בטווח שגיאה קטן לחלוקה עשרונית).
    *   **שאלה אמריקאית (Multiple-Choice):** בחירה מתוך 4 אפשרויות.
    *   **שאלת התאמה (Matching):** התאמת זוגות (למשל, בין פונקציה לנגזרת שלה) בממשק לחיצה צבעוני ונוח לנייד.
    *   **מיון לפי סדר (Drag & Drop Sort):** סידור שלבי פתרון מההתחלה לסוף באמצעות גרירה או שימוש בחצים (מותאם למקלדת ומובייל).
    *   **חלוקה לקטגוריות (Drag & Drop Slots):** מיון פריטים מבנק אל תוך קטגוריות מתאימות (למשל, מיון נקודות מעל, מתחת או על ישר).
*   **מערכת רמזים הדרגתית (Step-by-step Hints):**
    *   בכל ניסיון מענה שגוי, המשחק חושף רמז מפורט נוסף בתוך בועית מידע (Tooltip) ייעודית.
    *   הרמזים נכתבים בשילוב פורמולות מתמטיות ברורות כדי להוביל את התלמיד לפתרון צעד אחר צעד ללא תסכול.
*   **תצוגת נוסחאות מתמטיות עשירה (LaTeX Integration):**
    *   אינטגרציה מלאה עם מנועי הרינדור **KaTeX** (מהיר במיוחד) ו-**MathJax** (כגיבוי) המאפשרת הצגה של שברים, שורשים, מערכות משוואות ופונקציות בצורה גרפית מושלמת.
*   **שמירת התקדמות אוטומטית (Persistent State):**
    *   ההתקדמות (עולמות פתוחים, שלבים שהושלמו, נקודות ניסיון, כוכבים, רצף ותגים) נשמרת אוטומטית בדפדפן באמצעות `localStorage`. ניתן להמשיך מאותה נקודה בכל עת, או לאפס את ההתקדמות בלחיצת כפתור.
*   **מנוע אפקטים מתקדם (Audio-Visual Effects):**
    *   **מנוע קול:** ניסיון לטעון ולהשמיע קובץ סאונד של תרועת חצוצרה (`trumpet-fanfare.mp3`). אם הקובץ אינו קיים או נחסם על ידי הדפדפן, מופעל סינתיסייזר מבוסס **Web Audio API** המפיק תרועת חצוצרות עשירה ומסונתזת בזמן אמת.
    *   **מנוע קונפטי פרקטלים דיגיטליים (Digital Fractals):** אנימציית HTML5 Canvas מרהיבה של פיצוץ חלקיקים גיאומטריים זוהרים בצבעי ניאון (משולשים פרקטליים רקורסיביים, משושים קונצנטריים וקווים מצטלבים) הנעים בתנועה הרמונית (שילוב כוחות גליים של סינוס וקוסינוס) ודוהים בעדינות.

## 🛠️ כיצד להריץ את המשחק

המשחק מבוסס על קבצי Frontend סטטיים ומשתמש בטכנולוגיית **ES Modules** (מילות המפתח `import` ו-`export` בקבצי JavaScript). בעקבות מגבלות אבטחה של דפדפנים מודרניים (שגיאות CORS), **לא ניתן להריץ את הפרויקט על ידי פתיחה ישירה של קובץ ה-HTML מקומית (פרוטוקול `file://`)**. יש להריץ אותו באמצעות שרת מקומי.

### אפשרות 1: שימוש ב-VS Code Live Server (הכי פשוט)
1. פתח את תיקיית המשחק בתוך **Visual Studio Code**.
2. התקן את התוסף **Live Server** (של Ritwick Dey).
3. לחץ על הכפתור **Go Live** בפינה הימנית התחתונה של החלון.
4. המשחק ייפתח אוטומטית בדפדפן בכתובת `http://127.0.0.1:5500/index.html`.

### אפשרות 2: שימוש ב-Python (דרך טרמינל / PowerShell)
אם מותקן אצלך Python במחשב, תוכל להריץ שרת מקומי בשניות:
1. פתח את הטרמינל או ה-PowerShell בתיקיית הפרויקט.
2. הרץ את הפקודה הבאה:
   * **עבור Python 3:**
     ```bash
     python -m http.server 8000
     ```
   * **עבור Python 2:**
     ```bash
     python -m SimpleHTTPServer 8000
     ```
3. פתח את הדפדפן וגלוש לכתובת: `http://localhost:8000`.

### אפשרות 3: שימוש ב-Node.js (`http-server`)
1. התקן את הכלי הגלובלי באמצעות npm:
   ```bash
   npm install -g http-server
   ```
2. גש לתיקיית הפרויקט בטרמינל והרץ:
   ```bash
   http-server
   ```
3. פתח את הכתובת המוצגת בטרמינל בדפדפן.

## 🧪 שילוב ספריות חיצוניות (בתוך `index.html`)

על מנת לתמוך בעיצוב המודרני ובפורמולות המתמטיות, קובץ ה-`index.html` יכלול קישורים לטעינת הספריות הבאות (רצוי דרך CDN):
1.  **Tailwind CSS:** לעיצוב מהיר, גמיש ונקי בעל גריד רספונסיבי.
2.  **KaTeX (JS & CSS):** לרינדור מהיר ומדויק של פורמולות מתמטיות.
    *   יש לכלול גם את התוסף `contrib/auto-render.min.js` של KaTeX כדי לזהות אוטומטית סימני דולר (`$`) ברינדור.
3.  **MathJax:** (אופציונלי כגיבוי) לטעינת פורמולות אם KaTeX לא נטען.
4.  **Google Fonts:** פונטים נקיים ותומכי עברית כגון *Assistant* או *Heebo*.
